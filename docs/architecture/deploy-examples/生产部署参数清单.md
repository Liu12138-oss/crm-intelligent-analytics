- `crm-intelligent-analytics.service`

## 19. 上线前确认

- 所有敏感值是否已转入受控密码库，而不是继续保留在群聊或文档正文中
- 域名、证书、企业微信回调地址是否完全一致
- 正式环境是否允许服务器访问 AI 网关、CRM 登录 API、CRM 身份查询 API、联软标准 OpenAPI、CRM 数据库、企业微信 HTTP / WebSocket
- `.runtime` 是否已规划到持久化磁盘
- `3001` 是否只对本机或内网开放
- 日报部门启停、区域默认收件人和发送预览是否已由业务负责人签字确认
