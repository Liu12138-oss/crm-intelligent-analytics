# CRM智能分析系统架构说明

## 1. 架构目标

本实现以“企业微信问数入口 + Web 智能分析工作台 + 治理审计后台”的统一单仓结构落地一期 MVP，优先保证以下目标：

- 双入口共用同一套分析服务，保持口径一致。
- 除当前稳定的认证链路外，CRM 相关能力默认优先复用官方 API；分析问数、结果导出补查、目录同步、跟进写回等官方 API 未覆盖或需要兜底的场景，必须按对象、动作和确认链路进入受控数据库查询、受控内部接口或受控写回路径。
- 模板、最近查询、导出与审计事件统一落入应用库存储层。
- 前端严格遵循 `docs/prototype-ui/crm-intelligent-analytics-ui.pen` 与 `docs/prototype-ui/UI规范.md` 的三分布局。

## 2. 后端分层

- `database`：提供 CRM 数据源与应用库状态容器，当前实现使用内存版样例数据，后续可平滑替换为 MySQL。
- `auth`：负责 CRM 账号密码登录、企业微信扫码登录回调、本地会话建立、当前会话查询与统一鉴权装载。
- `analysis`：负责自然语言意图解析、补问、风险拦截、白名单校验、查询编译、结果归一化与一致性控制。
- `query-assets`：负责常用查询模板与最近查询记录。
- `sessions`：负责会话仓储、心跳保活、并发排队与能力快照。
- `governance`：负责治理策略查询与更新，前端由权限中心、连接策略等页面承载相关配置入口。
- `export`：负责导出额度校验、导出结果生成与审计记录。
- `audit`：负责审计事件查询。
- `wecom`：负责企业微信签名校验、身份映射、会话接入与流式预览编排。

## 3. 前端结构

- `LoginPage`：承载账号密码登录、企业微信扫码登录、登录失败提示与扫码引导。
- `AppShell`：负责 60px 顶栏、280px 左侧导航与主内容区布局。
- `AnalysisWorkbenchPage`：承载问数输入、权限摘要、AI 状态、结果摘要、图表、表格、常用查询与最近查询。
- `AnalysisResultDetailPage`：承载单次查询的完整摘要、筛选条件、图表解释、追问入口、流式结果块与导出动作。
- `QueryTemplatePage`：承载模板列表与模板新增入口。
- `ConnectionPolicyPage`：承载心跳、并发和会话阈值配置。
- `AuditEventPage`：承载审计概览卡、过滤器与事件列表。

## 4. 关键实现说明

- 当前后端采用 NestJS 单体 API，所有接口统一以 `/api/v1` 为前缀。
- `SessionAuthGuard` 优先通过 `crm_auth_session` Cookie 解析本地登录会话；仅在测试或本地联调场景下保留 `x-crm-user-id` 头降级能力。
- `CrmAuthService` 负责通过 CRM Open API 执行账号密码认证，并在后端建立本地会话，浏览器不直接持有外部 `user_token`；CRM 官方 API 文档统一参考地址为 `https://apidoc.weiwenjia.com/`。
- 除 `auth` 既有链路外，新增或改造 CRM 相关能力时，设计上必须先核对官方 API 是否可用；只有官方 API 不满足时，才允许落到受控自建接口或数据库路径。
- `analysis` 模块当前属于 CRM 官方 API 未覆盖的分析场景，因此继续通过白名单、权限和审计约束的受控数据库查询访问 CRM 数据源；涉及已批准写入动作时，必须通过声明过的受控执行器进入确认与审计链路。
- `LocalRuntimeConfigService` 会优先读取环境变量；若未显式配置，则自动从 `配置/codex配置/config.toml`、`配置/codex配置/auth.json` 和 `配置/数据库/数据库账号密码.txt` 装载 AI 网关与 CRM 数据源配置。
- 企业微信入口除签名外，还会校验消息来源、消息结构、发送者映射和渠道权限；测试环境默认允许 `test-signature`。
- 查询结果中的摘要、主视图、表格与导出共用同一 `consistencyToken`。
- 当前实现支持双模式运行：测试与无配置环境使用样例数据，真实配置可用时自动切换到 AI 网关 + CRM 官方 API（登录等已覆盖场景）/受控数据库查询与受控写入执行器（分析、补查、同步、写回等未覆盖场景）及企业微信原生映射同步。

## 5. 后续演进建议

- 将 `AppStorageService` 与 `CrmReadonlyService` 替换为真实 MySQL 连接。
- 为分析意图解析接入真实 AI 网关与结构化提示词。
- 增加更多主题域、真实图表渲染与 Playwright 端到端场景。
