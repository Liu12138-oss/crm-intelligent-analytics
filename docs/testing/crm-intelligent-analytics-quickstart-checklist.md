## 场景核对

- [ ] Web 智能分析工作台可打开并显示问题输入区、当前权限卡、AI 状态卡、常用查询区和最近查询区。
- [ ] 输入“本月各销售负责人新增商机金额排名”后，可返回标题、摘要、指标卡、图表和表格。
- [ ] 分析结果详情页会展示执行依据摘要、命中的语义资产标签，以及统一结果包中的来源说明或口径脚注。
- [ ] 输入缺少时间范围的问题后，系统先返回补问提示。
- [ ] 企业微信入口使用合法签名时可正常接入，非法签名会被拒绝。
- [ ] 最近查询可再次运行并生成新的查询记录。
- [ ] 智能合同审核页面可打开并只显示上传区与最近审核任务列表。
- [ ] 上传合法 `.docx` 合同后，系统会自动进入审核详情页并展示风险优先结果。
- [ ] 合同审核详情页与最近审核任务列表会展示审核标准版本、执行模式，以及 AI 补充审核状态提示（如有）。
- [ ] 当合同审核以默认 `DETERMINISTIC_ONLY` 规则快审完成时，详情页和最近审核任务列表会明确提示“规则快审”或等价语义，并说明当前仅覆盖已配置的明确判定项。
- [ ] 当合同审核因 AI 不可用、超时或返回非法结构而以 `DETERMINISTIC_ONLY` 降级快审完成时，详情页和最近审核任务列表会明确提示“降级快审”及原因，不会伪装成完整 AI 混合审核结果。
- [ ] 当 AI 补充审核处于 `PENDING`、`RUNNING`、`COMPLETED` 或 `FAILED` 时，详情页和最近审核任务列表会展示对应状态文案，并在完成后自动刷新到最新结果。
- [ ] 从合同审核详情页返回首页后，最近审核任务列表能看到刚创建的审核记录。
- [ ] 上传者本人可下载审核报告、结构化结果和带批注合同，且任务与产物元数据中的 `reviewBasis.packVersion`、`reviewBasis.packChecksumSummary`、`reviewBasis.executionMode` 保持一致。
- [ ] 未授权用户访问他人任务详情或下载地址会被拒绝。
- [ ] 导出权限不足时会被拦截并写入审计。
- [ ] 管理员在权限中心或连接策略页更新治理配置后，新策略可立即生效。
- [ ] 查询模板页面会展示语义资产最近发布版本与语义资产摘要列表。
- [ ] 发布新的语义资产后，对应问数结果详情和审计事件中可看到命中的语义资产信息。
- [ ] 审计中心可查看查询、导出和拦截事件。
