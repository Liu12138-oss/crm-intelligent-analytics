## ADDED Requirements

### Requirement: 生产环境必须提供企业微信官方 API 一键同步入口
系统必须提供基于企业微信官方通讯录 API 的一键同步入口，用于在生产环境发现目录未同步时立即触发目录刷新。

#### Scenario: 运维一键同步联软科技集团
- **WHEN** 生产环境发现机器人持续提示未绑定 CRM 身份
- **THEN** 运维必须能够通过一键同步入口立即刷新“联软科技集团”授权范围内的目录数据

### Requirement: 一键同步必须输出执行摘要
一键同步执行结束后，系统必须输出部门同步结果、成员同步结果、补齐详情结果和最新游标摘要。

#### Scenario: 一键同步成功完成
- **WHEN** 一键同步执行成功
- **THEN** 系统必须返回各资源类型的成功状态、条数和最新游标

### Requirement: 系统必须移除 EMM 目录同步入口
切换到企业微信官方 API 后，系统不得继续保留 EMM 目录同步脚本、EMM 同步内部入口或 EMM 配置读取逻辑。

#### Scenario: 部署新版本后不再接受 EMM 配置
- **WHEN** 系统完成官方 API 方案切换
- **THEN** 后端配置和脚本中不得再要求 `EMM_API_*` 相关参数
