## MODIFIED Requirements

### Requirement: 系统必须记录每轮企业微信目录同步任务状态
系统必须为每轮企业微信目录同步任务记录任务类型、开始时间、结束时间、成功状态、游标、分页统计和失败原因。该记录必须覆盖部门同步、成员同步和成员详情补齐三类任务，并明确记录同步来源为企业微信官方通讯录 API。

#### Scenario: 官方 API 部门同步成功
- **WHEN** 一轮部门同步全部成功完成
- **THEN** 系统必须记录成功状态、推进后的游标、同步条数和同步来源为 `wecom-official-api`

#### Scenario: 官方 API 成员同步失败
- **WHEN** 成员同步在某一阶段失败
- **THEN** 系统必须记录失败状态、失败原因、最近成功游标和同步来源为 `wecom-official-api`
