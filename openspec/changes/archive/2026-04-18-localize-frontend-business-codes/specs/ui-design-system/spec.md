## ADDED Requirements

### Requirement: Web 页面必须以中文业务标签展示内部代码

系统 MUST 在面向用户的 Web 页面中将后端返回的状态码、风险等级、审计事件类型、入口场景、目标工作流、fallback 原因、执行模式、数据来源、动作类型、模板状态、服务状态和视图类型转换为中文业务标签后展示。页面 MUST NOT 直接把全大写枚举、英文状态等级、kebab-case 原因码或内部工作流标识作为主要展示文案。

#### Scenario: 审计中心展示 AI 入口与 fallback 统计

- **WHEN** 审计接口返回 `WECOM_IDLE_MESSAGE`、`WECOM_DAILY_REPORT_ENTRY`、`active-conversation-flow-continue`、`LOW`、`critical` 或 `info`
- **THEN** 审计中心必须展示对应中文业务标签
- **THEN** 页面不得把这些内部代码作为主要可见文案直接展示给用户

#### Scenario: 分析结果详情展示执行状态与可执行动作

- **WHEN** 分析结果详情接口返回 `RETURNED`、`BLOCKED`、`PLAN_EXECUTION`、`GUARDED_READONLY_SQL` 或内部动作类型
- **THEN** 分析结果详情页必须展示中文状态、中文执行模式、中文执行来源和中文动作名称
- **THEN** 页面必须保留原有导出、追问和详情查看行为不变

#### Scenario: 治理页面展示策略与模板状态

- **WHEN** 治理策略或查询模板接口返回 `ACTIVE`、`INACTIVE`、`RANKING_TABLE` 或其他模板视图类型
- **THEN** 治理策略页和查询模板页必须展示中文状态和中文视图类型
- **THEN** 页面必须保留原有保存策略、创建模板和刷新数据行为不变

### Requirement: 前端必须提供统一业务代码展示出口

系统 MUST 在前端提供统一业务代码展示模块或同等能力，供页面和组件复用。该模块 MUST 按业务分类提供格式化函数，避免不同上下文的同名代码被错误翻译。页面 MUST 优先调用该统一出口，不得在多个页面重复维护同一枚举的中文映射。

#### Scenario: 页面需要显示风险等级

- **WHEN** 任一页面需要显示 `LOW`、`MEDIUM` 或 `HIGH`
- **THEN** 页面必须通过统一风险等级格式化出口获取“低风险”“中风险”“高风险”等中文标签
- **THEN** 页面不得在局部 template 中直接输出原始风险等级代码

#### Scenario: 页面需要显示审计入口或工作流

- **WHEN** 审计中心、分析详情或其他页面需要显示入口场景、目标工作流或 fallback 原因
- **THEN** 页面必须通过统一分类格式化出口获取中文标签
- **THEN** 同一代码在相同分类下的中文标签必须保持一致

### Requirement: 未知业务代码必须保留可诊断中文兜底

系统 MUST 对未知、未登记或新出现的业务代码提供中文兜底文案，并在兜底文案中保留原始代码。系统 MUST NOT 因未知代码而空白展示、静默隐藏或只显示英文原值。

#### Scenario: 后端新增尚未登记的审计事件类型

- **WHEN** 审计接口返回前端字典尚未登记的事件类型 `NEW_INTERNAL_EVENT`
- **THEN** 页面必须显示“未知事件类型（NEW_INTERNAL_EVENT）”或同等中文兜底
- **THEN** 页面不得只显示 `NEW_INTERNAL_EVENT`

#### Scenario: 可选字段为空

- **WHEN** 入口场景、工作流、fallback 原因或状态字段为空
- **THEN** 页面可以继续显示 `--` 或当前页面既有空状态中文文案
- **THEN** 页面不得把空值误判为未知代码

### Requirement: 页面级回归必须覆盖全部已注册路由页面

系统 MUST 以 `frontend/src/router/index.ts` 中注册的页面为检查清单，验证登录页、智能分析工作台、分析结果详情、合同审核工作台、合同审核详情、治理策略、查询模板、连接策略和审计中心不会裸露典型内部代码。相关组件测试和 E2E 测试 MUST 使用后端原始代码作为输入，并断言浏览器最终展示中文标签。

#### Scenario: 执行页面级本地化巡检

- **WHEN** 开发者完成业务代码中文化修复
- **THEN** 必须逐页检查所有已注册路由页面的用户可见文案
- **THEN** 必须确认截图样例中的 raw code 不再裸露在页面主体、表格、标签、卡片或提示中

#### Scenario: 后续新增页面或新增枚举

- **WHEN** 后续新增 Web 页面或新增需要展示的业务枚举
- **THEN** 必须同步扩展统一业务代码展示模块和测试样例
- **THEN** 不得通过页面内临时 `switch` 或直接插值绕过统一展示出口
