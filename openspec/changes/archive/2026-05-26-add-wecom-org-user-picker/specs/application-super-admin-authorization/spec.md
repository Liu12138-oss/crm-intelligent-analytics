## ADDED Requirements

### Requirement: 超级管理员授权人员必须支持按企业微信组织架构选择

超级管理员授权治理入口中的人员授权 MUST 支持通过企业微信组织架构人员选择器选择已映射 CRM 用户。系统 MUST 在选择器中展示企业微信组织上下文和 CRM 映射状态，并只允许保存有效 CRM 用户 ID 作为超级管理员授权主体。

#### Scenario: 管理员按组织架构授予用户超级管理员资格

- **WHEN** 管理员在超级管理员授权分区打开人员选择器并选择一名已映射 CRM 的企业微信成员
- **THEN** 系统必须将该成员对应的 CRM 用户 ID 加入超级管理员授权人员列表
- **THEN** 保存后下一次权限快照和权限预览必须识别该用户的超级管理员资格

#### Scenario: 未映射成员不能保存为超级管理员

- **WHEN** 管理员在超级管理员授权人员选择器中查看未映射 CRM 的企业微信成员
- **THEN** 该成员必须不可选
- **THEN** 系统不得将企业微信 userid 保存为超级管理员授权主体

### Requirement: 超级管理员授权人员搜索选择后必须清空关键词

超级管理员授权人员选择器 MUST 在管理员通过搜索结果选择人员后自动清空搜索关键词，且不得影响已选授权人员列表。

#### Scenario: 搜索选择授权人员后清空关键词

- **WHEN** 管理员在超级管理员授权人员选择器中输入关键词并选择人员
- **THEN** 搜索框必须恢复为空
- **THEN** 已选授权人员必须继续展示在已选列表中
