## ADDED Requirements

### Requirement: 分析工作台必须提供左侧可调常用查询资产区

Web 智能分析工作台 MUST 在桌面端提供左侧常驻常用查询资产区。该区域 MUST 遵循 `DESIGN.md` 的浅色主导、可信结果呈现、统一控件圆角、标签不换行和响应式要求。该区域 MUST 支持默认宽度、点击展开、点击收缩和拖拽调整宽度；宽度变化不得破坏右侧查询、结果表格、图表和操作按钮布局。

#### Scenario: 桌面端展示左侧常用查询资产区
- **WHEN** 用户在桌面端打开智能分析工作台
- **THEN** 页面必须展示左侧常用查询资产区
- **THEN** 右侧必须继续展示查询输入、结果和报告区域
- **THEN** 页面不得将常用查询默认隐藏在抽屉中

#### Scenario: 用户拖拽左侧资产区宽度
- **WHEN** 用户拖拽左侧常用查询资产区边界
- **THEN** 页面必须在受控最小宽度和最大宽度内调整布局
- **THEN** 右侧内容不得出现文字重叠、按钮挤压或表格不可读

### Requirement: 常用查询资产区必须支持标签页、筛选和大列表反馈

常用查询资产区 MUST 提供“我的模板”和“其它模板”标签页。两个标签页 MUST 支持关键词、标签和必要用户筛选。列表 MUST 支持虚拟滚动或等效大列表方案，并为加载中、加载失败、空状态、无权限、复制成功和执行失败提供中文反馈。

#### Scenario: 用户筛选模板
- **WHEN** 用户在左侧资产区输入关键词、选择分类或选择标签
- **THEN** “查询”按钮必须紧跟筛选控件
- **THEN** 用户按 Enter 必须执行同一次查询
- **THEN** 当前分析结果不得因筛选模板而被清空

#### Scenario: 模板列表为空
- **WHEN** 当前筛选条件下没有模板
- **THEN** 页面必须展示中文空状态
- **THEN** 空状态必须说明当前条件下没有匹配模板
- **THEN** 页面不得只显示“暂无数据”

#### Scenario: 模板列表加载失败
- **WHEN** 模板列表加载失败
- **THEN** 页面必须展示中文失败原因和重试入口
- **THEN** 页面不得只在控制台输出错误

### Requirement: 保存模板与复制模板必须提供明确交互反馈

保存为模板弹窗 MUST 包含模板名称、模板说明、标签、AI 推荐标签提示和保存 / 取消动作。标签控件 MUST 支持选择已有标签和自由输入新标签。保存、复制、校验失败和权限阻断 MUST 通过中文消息反馈，右上角瞬时提示默认 3 秒后自动消失，除非需要持续确认。

#### Scenario: 用户保存模板成功
- **WHEN** 用户填写保存模板表单并保存成功
- **THEN** 页面必须提示“模板已保存到我的模板”或等价中文文案
- **THEN** 提示必须自动消失或提供关闭入口

#### Scenario: 用户自由输入新标签
- **WHEN** 用户在标签控件中输入新的中文标签
- **THEN** 页面必须允许该标签作为模板标签保存
- **THEN** 标签不得换行挤压按钮或遮挡表单字段

#### Scenario: 保存模板失败
- **WHEN** 模板因无法复现 SQL、校验失败或权限不足保存失败
- **THEN** 页面必须先说明发生了什么
- **THEN** 页面必须给出下一步建议
- **THEN** 页面不得直接展示内部异常、SQL 片段或堆栈

### Requirement: 最近查询必须继续作为抽屉能力保留

智能分析工作台 MUST 保留最近查询抽屉入口。常用查询左侧常驻后，最近查询 MUST NOT 被强行合并进左侧常驻区。最近查询抽屉的再次运行、加载、失败、空状态和权限反馈 MUST 继续保持可用。

#### Scenario: 用户打开最近查询
- **WHEN** 用户点击最近查询入口
- **THEN** 页面必须打开最近查询抽屉
- **THEN** 最近查询列表必须展示个人历史记录
- **THEN** 左侧常用查询资产区不得被替换为最近查询

#### Scenario: 用户从最近查询再次运行
- **WHEN** 用户在最近查询抽屉中点击再次运行
- **THEN** 页面必须关闭或保持抽屉状态不影响主结果展示
- **THEN** 系统必须给出运行中、成功或失败反馈
