## ADDED Requirements

### Requirement: 系统必须控制生产运行态文件体积

系统 MUST 对生产运行态文件中的高频流水数据建立保留上限、归档和巡检机制，避免 `app-storage.json` 随 SQL 审计、请求记录或其它高频记录无限增长。运行态裁剪 MUST 只处理明确允许裁剪的数据集合，不得修改会话、模板、权限策略、企业微信上下文、合同审核产物或其它业务恢复所需状态。

#### Scenario: SQL 审计超过运行态保留上限

- **WHEN** 运行态中的 `sqlAuditRecords` 数量超过配置的保留上限
- **THEN** 系统必须仅保留最近的允许记录
- **THEN** 系统必须确保其它运行态集合的内容不被裁剪逻辑修改

#### Scenario: 生产归档前执行备份

- **WHEN** 运维脚本准备归档生产 `app-storage.json` 中的历史 SQL 审计
- **THEN** 脚本必须先生成完整备份
- **THEN** 归档过程必须只修改 `sqlAuditRecords`

#### Scenario: 运行态体积超过告警阈值

- **WHEN** `app-storage.json` 文件大小或 SQL 审计运行态数量超过配置阈值
- **THEN** 系统必须记录可检索告警或巡检结果
- **THEN** 告警内容不得包含数据库密码、Token、机器人密钥或其它敏感值

### Requirement: 关键 Web API 必须具备生产性能基线

系统 MUST 为登录、会话恢复、能力快照、模板列表、最近查询、历史查询、治理审计列表等关键 Web API 建立生产性能基线和回归门槛。性能观测 MUST 至少区分 Nginx 接入耗时、后端总耗时、上游 CRM Open API 耗时、CRM SQL 耗时和审计持久化耗时。

#### Scenario: 登录接口超过性能门槛

- **WHEN** 登录接口在生产环境的耗时超过配置的告警门槛
- **THEN** 系统必须能区分慢点发生在 CRM Open API、CRM 身份 SQL、审计持久化还是后端事件循环阻塞
- **THEN** 运维不得只能看到一个无法定位的总耗时

#### Scenario: 轻量接口被后端阻塞

- **WHEN** 能力快照、模板列表或最近查询接口出现秒级等待
- **THEN** 系统必须记录该接口的后端总耗时和关键内部阶段耗时
- **THEN** 系统必须能判断等待是否与运行态持久化、SQL 审计队列或身份解析有关

### Requirement: CRM 身份解析必须具备稳定性保护

系统 MUST 对 CRM 实时身份解析链路提供短时复用、并发去重、连接池恢复和降级观测能力。短时复用 MUST 用于减少同一会话在页面初始化期间的重复身份查询，不得绕过权限变更后的缓存失效，不得以长期授权快照替代当前实时权限。

#### Scenario: 同一会话并发请求身份上下文

- **WHEN** 用户登录后页面初始化并发请求多个受保护接口
- **THEN** 系统必须对同一会话的实时身份解析进行短时复用或并发去重
- **THEN** 系统不得为同一用户在短时间内重复发起等价角色和部门查询

#### Scenario: 身份查询连接池超时

- **WHEN** CRM 身份查询等待超过配置的身份查询超时阈值
- **THEN** 系统必须记录超时标签、尝试次数、阈值和连接池恢复结果
- **THEN** 系统必须避免单次连接池抖动持续拖慢后续轻量接口

#### Scenario: 权限策略发生变更

- **WHEN** 管理员更新角色权限、数据范围或模板治理策略
- **THEN** 系统必须失效相关能力快照和身份短缓存
- **THEN** 后续请求必须按更新后的实时权限重新判断

### Requirement: 生产性能优化必须支持灰度和回滚

系统 MUST 为运行态保留上限、SQL 审计异步写入、独立审计存储、身份缓存 TTL 和观测增强提供可配置开关或可回滚步骤。任何性能优化上线后如果影响现有登录、权限、审计、分析、合同审核或企业微信功能，系统 MUST 能恢复到上一稳定路径。

#### Scenario: 异步审计灰度出现异常

- **WHEN** 异步 SQL 审计写入出现队列积压、持续失败或审计记录缺失
- **THEN** 运维必须能通过配置关闭异步写入
- **THEN** 系统必须回退到同步写入加运行态保留上限的稳定路径

#### Scenario: 独立审计存储不可用

- **WHEN** 新增 SQL 审计独立存储不可用或迁移未完成
- **THEN** 系统必须保留旧运行态读取或兼容查询路径
- **THEN** 普通业务接口不得因为低风险只读 SQL 审计新存储不可用而失败

### Requirement: 生产排障不得暴露敏感信息

系统 MUST 在性能日志、巡检脚本、归档报告和告警中隐藏数据库密码、企业微信 Secret、Token、机器人密钥、完整连接串和其它敏感值。排障信息 MAY 包含环境变量名、接口路径、耗时、非敏感阈值、模块名和错误摘要。

#### Scenario: 输出生产性能巡检结果

- **WHEN** 运维执行生产性能巡检脚本
- **THEN** 脚本可以输出文件大小、记录数量、接口耗时、队列长度和非敏感环境变量名
- **THEN** 脚本不得输出明文密码、Token、Secret 或完整凭证 URL
