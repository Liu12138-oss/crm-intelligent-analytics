# 第一期主题清单与验收样例

## 首批落地主题

### 1. `owner-performance-ranking`

- 定位：负责人经营主题
- 落地优先级：P0
- 首批必选任务：
  - 负责人金额排名
- 首批可选任务：
  - 月度趋势
  - 阶段结构
  - 部门贡献
  - 高风险商机观察
- 首批区块：
  - 执行摘要
  - 关键指标
  - 趋势变化
  - 结构观察
  - 风险观察
  - 重点对象贡献
  - 明细结果
  - 行动建议

### 2. `region-operations`

- 定位：区域经营主题
- 落地优先级：P1
- 推荐任务：

  - 风险观察

### 3. `customer-operations`

### 4. `opportunity-risk`

- 定位：商机风险主题
- 落地优先级：P1
- 推荐任务：

### 5. `lead-funnel`

- 定位：线索漏斗主题
- 落地优先级：P2
- 推荐任务：

  - 趋势观察

## 首批验收样例

### 样例 A：负责人经营详细分析

- 输入：`最近一年各销售负责人新增商机金额排名，请做详细分析总结`
- 期望：
  - 命中 `owner-performance-ranking`
  - 报告深度为 `deep-dive`
  - 至少生成 5 个任务
  - 结果包包含 `summary / metric-strip / trend / distribution / risk / focus-list / detail-table / actions`
  - 结果包包含 `TOP1贡献占比`、`TOP3贡献占比`、`平均单笔商机金额`、`第一名领先第二名差距`

### 样例 B：负责人主题轻量复合问法

- 输入：`本季度各销售负责人新增商机金额排名和趋势`
- 期望：
  - 命中 `owner-performance-ranking`
  - 报告深度为 `snapshot`
  - 至少生成主排名和趋势两个任务
  - 结果包仍保持统一区块结构

### 样例 C：风险主题问法

- 输入：`最近三个月有哪些高风险商机`
- 期望：
  - 命中 `opportunity-risk`
  - 报告深度至少为 `standard`
  - 不得退化成普通单表明细
